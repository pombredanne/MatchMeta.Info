MatchMeta.Info
==============
MatchMeta.Info uses Python to create and match a SQLite database for unique full paths from XP, VISTA, WIN2K3, WIN7, WIN2K8, WIN8, WIN2K12 and WIN10 obtained from F-Response Flexdisk.
